Ce dossier regroupe tous les guides pratiques pour obtenir un résultat concret avec Daznode et le Lightning Network.

## Format standard à respecter

- **Objectif clair** (1 phrase)
- **Prérequis** (liste)
- **Étapes numérotées** (actions directes)
- **Code/commandes** (si applicable)
- **Résultat attendu**
- **Bugs connus / erreurs fréquentes**

Chaque tutoriel doit suivre ce modèle pour garantir efficacité, clarté et reproductibilité.

> Utilisez le style direct : impératif, phrases courtes, voix active, zéro blabla.

## Objectif

Vous trouverez ici des guides étape par étape pour vous aider à utiliser efficacement la plateforme Daznode, configurer votre nœud Lightning, exploiter la technologie RAG, et bien plus encore.

## Structure recommandée pour chaque tutoriel

Chaque tutoriel doit contenir :
- Un titre clair
- Un résumé des objectifs
- Les prérequis
- Les étapes détaillées, numérotées
- Des exemples concrets (captures d'écran, extraits de code, etc.)
- Une section de dépannage ou FAQ
