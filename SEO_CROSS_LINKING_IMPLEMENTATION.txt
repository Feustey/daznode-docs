
## ⚙️ Configuration Pages

### **Pour Activer les CTA sur une Page**
```yaml
---
layout: docs-with-cta.njk

topic: "Lightning Network"
category: "lightning-network"
showRoi: true

  - name: "Solution Name"
    url: "https://dazno.de/solution"
    description: "Description solution"
conversionBridge:
  - name: "CTA Name"
    url: "https://dazno.de/cta"
    description: "Description CTA"
---
```
