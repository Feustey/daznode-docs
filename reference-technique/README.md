# Référence technique

Ce dossier regroupe toute la documentation technique destinée aux développeurs et intégrateurs.

## Format standard à respecter

- **Endpoint/méthode**
- **Champs obligatoires**
- **Exemples de requêtes/réponses**
- **Limites connues**
- **Liens croisés vers les guides**

Chaque page doit suivre ce modèle pour garantir la clarté et l'uniformité.
